# Travel App – Salesforce Admin Project

## 📌 Project Overview
Developed a custom Salesforce Lightning Application to manage employee travel requests and approval workflows.

This project was built in a Trailhead Playground as a hands-on Salesforce Admin implementation.

---

## 🚀 Features Implemented

### 🔹 Custom Objects
- Travel Request
- Expense
- Department (if created)

### 🔹 Custom Fields
- Travel Start Date
- Travel End Date
- Estimated Cost
- Approval Status
- Manager Name

### 🔹 Automation
- Built Flow to automate travel approval process
- Configured Approval Process for request validation
- Created Validation Rules to prevent incorrect submissions

### 🔹 Security
- Configured Profiles / Permission Sets
- Role hierarchy for manager approvals

### 🔹 Reporting
- Created Reports for Travel Status Tracking
- Designed Dashboard to monitor approvals and expenses

---

## 🛠 Tools & Technologies Used
- Salesforce Lightning Experience
- Flow Builder
- Object Manager
- Reports & Dashboards
- Approval Processes
